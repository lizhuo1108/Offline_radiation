The repository includes the modified RRTMG files written in Fortran, which should be put back into the CESM model to get the input variables. After you get the input variables, put them as the input to the RRTMG and do the calculation (Offline_RRTMG.ipynb). The modified Fortran files are for CAM4. 

Please compare the online and offline result to make sure that the offline calculation is right.
